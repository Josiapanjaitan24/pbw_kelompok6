===========================================
TROUBLESHOOTING
===========================================

Problem: Produk tidak muncul di home
Solusi:
  1. Buka Console (F12) â†’ Check error
  2. Pastikan get_products.php accessible
  3. Refresh page (Ctrl+F5)
  4. Cek database - ada data di tabel produk?

Problem: Foto tidak tampil
Solusi:
  1. Pastikan folder uploads/produk/ ada
  2. Cek foto tersimpan di folder itu
  3. Cek path di database (kolom foto)

Problem: Login admin error
Solusi:
  1. Pastikan user punya is_admin = 1
  2. Buka http://localhost/project/back-end/manage_users.php
  3. Klik "Set as Admin" untuk user Anda

Problem: Form tidak tersimpan
Solusi:
  1. Cek server error di browser console
  2. Pastikan kolom spesifikasi ada di database
  3. Jalankan ALTER TABLE (lihat SETUP_GUIDE.txt)
